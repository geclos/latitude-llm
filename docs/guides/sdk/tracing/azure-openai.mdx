## Usage

To trace Azure OpenAI calls with Latitude, you need to install the Azure OpenAI
SDK and initialize Latitude with your API key:

```typescript
import * as AzureOpenAI from '@azure/openai'
import { Latitude } from '@latitude-data/sdk'

new Latitude('your-api-key-here', {
  telemetry: {
    modules: {
      azureOpenAI: AzureOpenAI,
    },
  },
})
```
