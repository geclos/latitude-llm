## Usage

To trace Cohere calls with Latitude, you need to install the Cohere SDK and
initialize Latitude with your API key:

```typescript
import * as Cohere from 'cohere-ai'
import { Latitude } from '@latitude-data/sdk'

new Latitude('your-api-key-here', {
  telemetry: {
    modules: {
      cohere: Cohere,
    },
  },
})
```
