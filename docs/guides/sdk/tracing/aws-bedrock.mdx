## Usage

To trace AWS Bedrock calls with Latitude, you need to install the AWS Bedrock SDK
and initialize Latitude with your API key:

```typescript
import * as Bedrock from '@aws-sdk/client-bedrock-runtime'
import { Latitude } from '@latitude-data/sdk'

new Latitude('your-api-key-here', {
  telemetry: {
    modules: {
      bedrock: Bedrock,
    },
  },
})
```
