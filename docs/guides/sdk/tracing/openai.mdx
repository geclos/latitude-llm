## Usage

To trace OpenAI calls with Latitude, you need to install the OpenAI SDK and
initialize Latitude with your API key:

```typescript
import OpenAI from 'openai'
import { Latitude } from '@latitude-data/sdk'

new Latitude('your-api-key-here', {
  telemetry: {
    modules: {
      openAI: OpenAI,
    },
  },
})
```
